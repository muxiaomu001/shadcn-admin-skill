# Shadcn-Admin Skill

Claude Code Skill for developing admin dashboards based on [shadcn-admin](https://github.com/satnaing/shadcn-admin) project.

## Features

- Page generation and layout patterns
- Component development with shadcn/ui
- Data table CRUD with TanStack Table
- Form handling with React Hook Form + Zod
- Automation scripts for scaffolding

## Installation

Copy the skill files to your project's `.claude/skills/shadcn-admin/` directory.
